# UMAS-Pilot: Uyghur-Script Mongolian Annotation Scheme — Pilot Dataset

[![License: CC BY-SA 4.0](https://img.shields.io/badge/License-CC%20BY--SA%204.0-lightgrey.svg)](https://creativecommons.org/licenses/by-sa/4.0/)

## Overview
This repository contains the pilot annotation dataset for the  
**UMAS (Uyghur-Script Mongolian Annotation Scheme)**, as described in:

> Nan, Z.; Batu, B.; Tala; Zhang, G.; Qiao, C.; Chen, H.  
> "A Hierarchical Polygon Annotation Scheme and Pilot Dataset for  
> Low-Resource Vertically Written Scripts: A Case Study on  
> Uyghur-Script Mongolian."  
> *Information* 2026. https://doi.org/10.3390/infoxx

**Project funding:** Inner Mongolia Autonomous Region Key R&D Program,  
Grant No. 2023YFSW0004.

## Repository Contents
| File | Description |
|------|-------------|
| `labels/1dd8cd11-02.a.txt` | Pilot annotation — YOLO polygon format (561 entries) |
| `annotations/1dd8cd11-02.a.json` | Pilot annotation — COCO JSON format |
| `config/classes.txt` | UMAS 54-category taxonomy (class IDs 0–53) |
| `config/notes.json` | Label Studio category metadata |
| `scripts/analysis.py` | Character frequency & polygon complexity analysis |
| `docs/UMAS_Guidelines_v1.0.pdf` | Full annotation guidelines (28 pages) |
